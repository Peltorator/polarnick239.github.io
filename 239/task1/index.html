Задание 1. Клеточный автомат на Android

Настройка окружения, Android Studio:

  • скачать Android Studio: https://developer.android.com/studio/index.html
  • установить, оставив все по дефолту (галочки стоят на всех трех пакетах)
  • Запустить Android Studio
  • "I do not have a previous version ..." -> Ok
  • "Standard" -> Next
  • галку на Android Virtual Device (если хочется эмулятор) -> Next
  • Finish, ждать пока грузится 600 Мб

Git:

  • скачать git клиент https://git-for-windows.github.io/
  • установить его (всё оставлять по умолчанию)
  • запустить Android Studio
  • открыть настройки Android Studio (либо Configure->Settings если вы видите маленькое окно, либо File->Settings если вы видите запущенную Android Studio)
  • в настройках Android Studio: Version Control -> Git
  • Path to Git executable: указать git.exe файл, который установлен примерно по такому пути: C:\Program Files\Git\bin\git.exe
  • Нажать справа кнопку "Test", должно показаться окно с версией git
  • Закрываем настройки, нажав Ok

Задание "Клеточный автомат"

  • Запустить Android Studio
  • Check out project from Version Control->Git (НЕ github)
  • Git repository URL: https://github.com/PolarNick239/AndroidBenchmarks
  • Если пишет красным "The parent path C:\...\StudioProjects must exist" - создайте такую папку, либо укажите другую Parent Directory
  • нажать Clone
  • would you like to create a Studio project for the sources you have been checked ...? -> нажать Yes
  • create project from existing sources -> Next -> и еще раз Next -> и еще раз Next -> и еще один терпеливый раз Next -> ну и чего уж там еще один Next -> вообщем Next до упора
  • закрываем окошко Tip of the day -> Close

Уровни сложности:

  • Easy (если обычно вас напрягает информатика, или сделать Medium вы не успеваете)
  • Medium (вариант по умолчанию, на другой уровень сложности в любой момент можно переключится, скопировав то что уже написали)
  • Nightmare (на самом деле это Medium, но еще одна сложная технически задачка добавлена, поэтому когда будет сделан Medium - можно попробовать Nightmare, скопировав уже сделанную часть

Настройка проекта:

  • Снизу слева нажать на "Unregistered VCS root detected" -> нажать синюю надпись Add Root
  • справа снизу нажать на "Git: master" -> origin/easy -> checkout as new loca branch -> Ok
  • Снизу нажав на TODO можно увидеть перечень заданий

Android NDK:

  • Скачать Android NDK:
  • либо Project Structure->SDK Location-> снизу нажать на "Download" Android NDK
  • либо руками скачать отсюда https://developer.android.com/ndk/downloads/index.html а затем в Project Structure->SDK Location-> снизу указать путь на этот распакованный архив

Чтобы запускать:

  • Run->Edit configurations
  • Зеленый плюс->Android App
  • Module: выбрать main (единственный вариант)
  • Target: Emulator (тут можно указать USB, если захочется проверить на своем телефоне)
  • Prefer Android Virtual Device: выбрать "Nexus 5x ..." (единственный вариант)
  • Ok
  • теперь можно запускать сверху зеленым треугольником (как кнопка Play в плеерах)
